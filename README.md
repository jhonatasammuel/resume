# 🚀 Web Resume & CSS Art Lab

Este projeto funciona como um currículo digital e um laboratório de evolução em Front-End. Ele demonstra minha jornada técnica, partindo da estruturação de dados até a estilização avançada de interfaces.

### 🛠️ Estrutura do Projeto
- **Currículo (Home)**: Desenvolvido em **HTML5 puro**, focando em acessibilidade, organização de informações profissionais, links de contato e formação acadêmica.
- **Portfólio / Recriação**: Uma seção dedicada a demonstrar habilidades em **CSS3**. Ao acessar o link de "Recriação", você encontrará uma obra de arte geométrica (estilo Mondrian) construída inteiramente com **divs, containers e estilização CSS**.

### 🎨 O Diferencial: CSS Art
Diferente de usar uma imagem estática, a recriação da obra foi feita através de:
- **Box Model**: Dimensionamento preciso de blocos e bordas.
- **Layout de Containers**: Organização de divs para compor uma interface visual complexa.
- **Marcação e Estilo**: Separação clara entre a estrutura HTML e o design CSS.
